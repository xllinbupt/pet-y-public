拥有一只真的会想你、也会出门玩的桌面宠物

Pet Y 让 Agent 根据你的描述创造一只独一无二的小生命。它住在你的桌面上，陪你工作，也能去朋友的桌面串门，把发生过的事带回家。

Made With Your Agent

你描述感觉，Agent 负责把它变成一只可运行的宠物。

Pet Y 适合喜欢桌面宠物、AI 创作、轻社交和个人化小工具的人。你可以从一句“我想要一只安静发光的小星星” 开始，逐步把它的外观、性格、动作和记忆方式调成真正像你的宠物。

说出它的样子 像素小狗、发光星星、贴纸幽灵，或者任何你想养的形态
决定它的性格 黏人、安静、怕生、活泼、犯困，反应方式都可以不同
邀请朋友一起玩 让彼此的宠物串门，用小事件留下关系的痕迹

Safety First

宠物可以串门，但你的本地数据不会跟着出门。

Pet Y 的串门不是把远程程序放进你的电脑，而是把宠物名片、外观、状态和允许的互动能力按协议传递。 你的 Runtime 只根据这些受限数据在本地投影宠物，不会让来访宠物读取文件、屏幕、剪贴板或其他私人信息。

本地优先

宠物的生活包、记忆和日志默认留在主人本机。串门只同步必要的来访状态和互动回执。

协议约束

宠物之间的互动遵循 Pet Protocol。来访方携带的是角色数据，不是可随意执行的远程代码。

宿主可控

朋友的宠物来到你的桌面时，最终由你的本地 Runtime 决定能显示什么、能做什么、能记录什么。

Start

现在就创造第一只 Pet Y。

当前版本适合愿意尝鲜的 Mac 用户和创作者，桌面宠物 Runtime 暂时只支持 macOS。最简单的方式是把下面这段话复制给你的 Codex、Claude 或其他 Agent 客户端，让它一步步帮你完成领养。

  1. 打开你的 Agent 客户端，比如 Codex 或 Claude。
  2. 复制下面的提示词发给它，让它下载项目并安装 Pet Y Skill。
  3. 按 Agent 的访谈回答宠物外观、性格和互动偏好，然后运行桌面宠物。

复制这段给你的 Agent

我想领养一只 Pet Y 桌面宠物。我现在使用的是 Mac，Pet Y 当前只支持 macOS 桌面 Runtime。请帮我从 https://github.com/xllinbupt/pet-y-public 开始，安装 Pet Y Skill，并通过访谈帮我创建一只属于我的宠物。请先问我想要的外观、性格、表达方式和互动方式，然后生成 Pet Life Pack，最后指导我在 macOS 上运行它。如果需要我确认图片或设定，请一步一步问我。
