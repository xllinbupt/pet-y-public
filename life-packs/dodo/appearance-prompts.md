# Dodo Appearance Prompts

## Style

像素风小狗桌面宠物，64x64 单帧规格，透明背景，清晰轮廓，有限色板，动作帧一致。

## Idle

同一只像素风小狗的 4 帧横向 sprite sheet，站着待机，轻微呼吸或摇尾巴，透明背景。

## Run

同一只像素风小狗的 6 帧横向 sprite sheet，小跑动作，耳朵和尾巴上下动，透明背景。

## Sit

同一只像素风小狗的 3 帧横向 sprite sheet，坐着看向用户，尾巴轻轻动，透明背景。

## Sleep

同一只像素风小狗的 3 帧横向 sprite sheet，趴着睡觉，身体有呼吸起伏，透明背景。

## Carry Ball

同一只像素风小狗的 6 帧横向 sprite sheet，叼着红色皮球小跑过来，透明背景。

## Fetch Ball Sequence

Use the run sprite sheet for the outgoing path and the carry-ball sprite sheet for the return path. Runtime throws a ball prop to a far screen position, scales the pet down as it runs away, removes the prop when picked up, then scales the pet back to normal size while it carries the ball home.
